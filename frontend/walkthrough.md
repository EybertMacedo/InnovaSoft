## Features Implemented

1.  **Tech Stack**: Vue 3 (Composition API), Vite, Tailwind CSS, Lucide Icons.
2.  **Design**: "Enterprise AI" / "Industrial Futurism" theme with dark mode, glassmorphism, and animated gradients.
3.  **Components**:
    *   **Hero Section**: Animated entry, futuristic vibe.
    *   **Projects Grid**: Displaying the 7 requested projects with status badges.
    *   **Founder Profile**: Bio and tech stack for Alexis.
    *   **Interactive Demo Center**:
        *   **Chat Simulator**: WhatsApp-style bot with typing effect.
        *   **Vision Simulator**: YOLOv8 mock with scanning animation and bounding boxes.
        *   **Dashboard Simulator**: Real-time metrics with animated counters and charts.

## How to Run

1.  Navigate to the project directory:
    ```bash
    cd innovasoft-landing
    ```

2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at the URL shown (usually `http://localhost:5173`).

## File Structure

*   `src/App.vue`: Contains the entire application logic and UI as a Single File Component.
*   `tailwind.config.js`: Custom configuration for the theme.
*   `src/style.css`: Tailwind directives and global animations.
